# 测试文章

欢迎使用 wxmd-cli！这是一篇用于测试 CLI 功能的示例文章。

## 功能特性

- **本地排版**：无需服务器，直接转换 Markdown 为 HTML
- **多种样式**：支持技术、优雅、纽约时报等多种风格
- **API 集成**：支持创建分享链接
- **结构化输出**：面向 Agent 的 JSON 格式

## 代码示例

```javascript
// 示例代码
const greeting = "Hello, wxmd-cli!";
console.log(greeting);
```

## 表格展示

| 命令 | 功能 | 需服务器 |
|------|------|---------|
| typeset | Markdown 排版 | 否 |
| share create | 创建分享 | 是 |
| share get | 获取分享 | 是 |
| styles list | 列出样式 | 否 |
| doctor | 环境检查 | 部分 |
